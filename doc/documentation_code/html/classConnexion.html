static Connexion::getConn ( )
static

Obtient l'instance de connexion à la base de données.

Lit la configuration de la base de données à partir d'un fichier JSON et établit une connexion à la base de données en utilisant MySQLi. Si le fichier JSON est absent ou comporte des configurations manquantes, il utilise des valeurs par défaut.

Returns
mysqli L'instance de connexion à la base de données MySQLi.
